# FreeTablets GitHub Pages Support Site

This package is a static support site designed for GitHub Pages.

## Included
- Large homepage
- 4 article pages
- About, Editorial Policy, Contact, Privacy Policy, Terms
- Structured data
- Sitemap and robots
- Canonical tags mapped to the main domain

## Deploy
1. Upload the files to the `freetablets.github.io` repository root
2. Keep `.nojekyll` in place
3. Commit and publish through GitHub Pages

## Canonical setup
This package uses cross-domain canonicals pointing to `https://freetabletgov.click` and related main-domain paths.

## Replace before final launch
- Placeholder contact emails on the Contact page
- Any branding text you want changed
